Grocery Bud (CRUD App) <br>

A simple CRUD Grocery List application built using JavaScript. <br>
Supports adding, editing, deleting, and marking grocery items as completed. <br>

 Features: <br>

Add grocery items <br>

Edit item names <br>

Delete items <br>

Mark items as completed <br>

Save data using Local Storage <br>

Tech Stack: <br>

HTML <br>

CSS <br>

JavaScript <br>

Run Project: <br>

Open index.html in your browser <br>
